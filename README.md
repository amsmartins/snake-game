## Como Jogar:

* Use as setas <strong> ↤ ↥ ↦ ↧</strong> do teclado para se mover, controlar a cobra.
* Evite colidir nas paredes ou na calda da cobra. Senão, <strong>GAME OVER!</strong>
* Aperte <strong>F5</strong> para jogar de novo.
